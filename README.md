# 🎮 Video Game Sales Prediction

A Machine Learning project that predicts global video game sales using Linear Regression. The project includes data preprocessing, model training, evaluation, and an interactive Streamlit dashboard for exploring game sales across different genres and publishers.

---

## Features

- Data Cleaning
- Missing Value Handling
- Linear Regression Model
- Model Evaluation (MSE & R²)
- Interactive Dashboard
- Data Visualization

---

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit
